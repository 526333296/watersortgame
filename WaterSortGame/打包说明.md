## 第一步：安装 Android Studio

1. 访问 https://developer.android.com/studio 下载
2. 安装时勾选 "Android Virtual Device"（可选，有实体手机可跳过）
3. 首次启动会自动下载 Android SDK（需要科学上网或等待）

---

## 第二步：用 Android Studio 打开项目

1. 打开 Android Studio → **File → Open**
2. 选择本文件夹（`WaterSortGame` 文件夹）
3. 等待 Gradle 同步完成（底部进度条跑完，首次约 5-10 分钟）

---

## 第三步：编译 APK

### 方法 A：Debug APK（直接安装测试，推荐）

菜单 → **Build → Build Bundle(s) / APK(s) → Build APK(s)**

等待编译，完成后点弹窗里的 **"locate"** 找到 APK 文件：
```
app/build/outputs/apk/debug/app-debug.apk
```

### 方法 B：Release APK（正式发布版，稍小）

菜单 → **Build → Generate Signed Bundle / APK**
→ 选 APK → 创建签名密钥 → 编译

---

## 第四步：安装到手机

### 安卓手机（USB 连接）
1. 手机 → 设置 → 关于手机 → 连续点"版本号"7次 → 开启"开发者模式"
2. 开发者选项 → 开启 **USB 调试**
3. 连接电脑，Android Studio 顶部下拉选择你的手机
4. 点绿色 **▶ Run** 按钮直接安装运行

### 安卓手机（传文件安装）
1. 把 APK 文件传到手机（微信、QQ 或 U 盘均可）
2. 手机设置 → 安全 → 允许安装来自未知来源的应用
3. 手机文件管理器里点 APK 安装

### 安卓电视
1. 电视设置 → 系统 → 开发者选项 → 开启 USB 调试（或 ADB over WiFi）
2. 或将 APK 拷贝到 U 盘插入电视，用电视文件管理器安装
3. 安装完成后在应用列表找到 **"彩色倒水游戏"**

## 电视遥控器操作说明

| 按键 | 功能 |
|------|------|
| 方向键 ← → | 移动选中水瓶（左右） |
| 方向键 ↑ ↓ | 移动选中（5格跳跃） |
| 确认键（中间） | 点击/选择当前高亮项 |
| 返回键 | 返回上一页面 |

## 常见问题

**Q：Gradle 同步失败 / 超时？**
A：需要下载 Gradle 依赖，确保网络通畅。也可在 Android Studio 设置里配置代理。

**Q：找不到 SDK？**
A：Android Studio → File → Project Structure → SDK Location，指向你安装的 SDK 路径。

**Q：电视安装后找不到应用？**
A：在电视设置 → 应用管理 里查找，或者应用商店有"全部应用"入口。

**Q：游戏进度保存在哪里？**
A：使用 WebView 的 localStorage，卸载 APP 才会清空。
